# Pasos para iniciar tu entorno de desarrollo

* 1. Abrir Docker Desktop
* 2. Correr el compose --> docker-compose up --build
Si da error por puerto, ir a Inicio / buscar Servicios / detener MySQL80
* 3. Ingresar al Dev Container en VS Code --> ctrl + shft + p / Dev Containers: Reopen in Container

# Para cerrar todo:

* 1. Cerrar el Dev Container --> ctrl + shft + p / Reopen Folder Locally
* 2. Cerrar el compose --> docker-compose down
* 3. Cerrar Docker Desktop
